# Smart Banking Management System

A secure banking web application developed using Java Servlets, JSP, JDBC and PostgreSQL.

## Features

- User Registration & Login
- Create Multiple Bank Accounts
- Deposit Money
- Withdraw Money
- Transfer Funds
- Beneficiary Management
- Transaction History
- Complaint Management
- Notification System
- Admin Dashboard
- Account Closure Requests

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- PostgreSQL
- Apache Tomcat
- HTML
- CSS
- JavaScript
